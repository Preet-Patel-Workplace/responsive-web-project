# Responsive Login UI

## Overview
A modern and responsive login user interface built using HTML5 and CSS3, focused on clean design, accessibility, and user experience.

## Features
- Semantic and accessible HTML structure
- Responsive layout optimized for desktop and mobile
- Modern UI styling with CSS variables
- Animated background and interactive elements
- Password visibility toggle using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (basic DOM manipulation)

## Key Learnings
- Designing clean and reusable UI components
- Implementing responsive layouts with Flexbox
- Managing UI states and interactions
- Writing structured, maintainable CSS# responsive-web-project
